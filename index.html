Creating your website

To create a website you need to have content to fill the site with

All websites are created in HTML which stands for hyper text markup language

In order to create a HTML file use notepad but when you save it make sure you call it index.html and save it as All files.

When you try to open the file it should open your browser and your website should be displayed. Although it is in your internet browser, your website is not yet on the internet.

If you want to edit your html code again, you have to right click it and select open with notepad.

If you want to create CSS or Javascript to your website you have to do the same process but name them style.css and main.js respectively

To connect your HTML with your CSS you have to add the code <link href="/Website/style.css" type="text/css" rel="stylesheet" /> to the header part of your HTML

To connect your HTML with your Javascript you have to add the code <script src="/Website/main.js"><script>

The src part of the Javascript link and href part of the CSS link must be replaced by the path to your own files.

You can also add an image next to the title of the webpage known as a favicon by using similar code: <link rel='shortcut icon' type='image/x-icon' href='/Website/favicon.ico' />.

Keep in mind that when you deploy your website you might have to change the file paths.

Using your own images

In order to use your own images on your website you must upload your images to an image hosting site and use the url given to you in your html.

Deploying your website

To deploy my website I used Github pages.

In order to deploy a website with Github you must create an account then create a new repository to store your website information.

You can select the option to initialize the repository with a README file, which you can delete later.

OR

You can use command prompt to create a Git repository for your website directory and connect that repository to the repository in Github like I did.

Because the school prevents students from accessing command prompt, I had to use Codecademy's lesson to initialize my Git repository, which lets you access an instance of command prompt in their servers.(I think).

Once you have done that you can create, upload, and edit files within Github.

In order to actually deploy your website to the internet go to settings and scroll down to the section called Github pages.

From there set the source to the master branch and save.

Congratulations, your website should be up and running at the default domain given to you.

You can also change the domain but I have not learned how to yet.
